## 安装与系统要求

**适用场景**：第一次在电脑上安装知聊，或从旧版本升级。

**使用教程**

第一步：确认本机微信为 **4.0 及以上**版本（知聊仅支持新版微信数据目录结构）。

第二步：打开 [WeFlow-Release 下载页](https://github.com/NewbieCheng/WeFlow-Release/releases/latest)，下载对应系统的安装包：

| 平台 | 安装包 |
|------|--------|
| Windows 10+ x64 | `ZhiLiao-*-Setup.exe`（桌面快捷方式「知聊」） |
| macOS Apple Silicon | `WeFlow-*.dmg`（应用包名 WeFlow.app，界面显示「知聊」） |
| Linux x64 | `.AppImage` 或 `.tar.gz` |

第三步：按安装向导完成安装；**Windows** 若后续提示无法读取微信进程，可 **右键「知聊」快捷方式 → 以管理员身份运行**。

第四步：首次启动后进入下一节「连接数据库」；macOS 密钥问题见 [MAC-KEY-FAQ.md](MAC-KEY-FAQ.md)。

![安装-下载 Release](images/01-install-download.png)

![安装-完成启动](images/02-install-launch.png)

## 首次启动与连接数据库

**适用场景**：安装后第一次使用，或更换微信账号/数据目录后重新连接。

**使用教程**

第一步：启动 WeFlow，在引导页（欢迎页）阅读协议并同意。

第二步：在 **数据库路径** 处点击 **自动检测**；若失败，手动选择微信数据根目录（Windows 一般为 `文档\xwechat_files`，macOS 为 `2.0b4.0.9` 等目录，界面有示例提示）。

第三步：选择当前要分析的 **微信账号（wxid）**；多账号用户可在 **账号管理** 中切换。

第四步：按提示完成 **数据库密钥** 获取（部分环境需等待扫描或按 FAQ 操作）；连接成功后进入主界面，侧边栏各功能可用。

第五步：若仅看到首页/设置而无法进入聊天，说明尚未连库成功，请返回引导页重试。

![连库-选择路径](images/03-connect-db-path.png)

![连库-连接成功](images/04-connect-success.png)

## 1、查看与搜索聊天记录

**适用场景**：在本地浏览某好友或群聊的历史消息，查看图片、语音等。

**使用教程**

第一步：点击侧边栏 **聊天**。

第二步：在左侧会话列表选择好友或群聊；可使用顶部搜索。

第三步：右侧查看消息流；需要时点 **跳转日期** 定位到某一天。

第四步：图片、视频、语音等内容在支持的情况下会自动解密显示；部分消息可 **修改** 或 **删除本地记录**（仅影响 WeFlow 本地展示，不修改微信服务器）。

![聊天-会话列表](images/07-chat-sessions.png)

## 2、聊天整理 / SOP（特供）

**适用场景**：把指定 **群聊** 在某一时间段内的记录整理为课程资料，导出 **Word（docx）**、Markdown 或 JSON，可选 AI 改写为讲义。

**使用教程**

第一步：在 **首页** 点击 **聊天整理 / SOP**，或访问 `/sop` 页面。

第二步：在群聊列表中 **搜索并选择目标群**（仅群聊）。

第三步：设置 **时间范围**（全部 / 最近 7 天 / 最近 30 天 / 自定义日期）。

第四步：选择 **输出模式**：

| 模式 | 说明 |
|------|------|
| **仅归档** | 直接导出聊天记录与图片 |
| **归档 + AI 摘要** | 在归档基础上用 AI 生成课程文档（需先配置 AI） |

第五步：选择 **导出格式**（docx / markdown / json），点击开始整理；完成后可 **打开输出目录** 或打开生成的文件。

第六步：输出默认在 `文档\WeFlow\ai-sop\` 下；提示词、分页大小等在 **设置 → AI 设置 → 聊天整理** 中调整。

**前置条件**：**设置 → AI 设置 → 基础配置** 中已填写可用的 API 地址、Key 与模型（与「AI 见解」共用）。

![SOP-选择群聊](images/08-sop-select-group.png)

![SOP-导出完成](images/09-sop-done.png)

## 3、导出中心

**适用场景**：将会话聊天记录导出为 HTML、JSON、Excel、ChatLab 等格式，适合归档或导入其他工具。

**使用教程**

第一步：点击侧边栏 **导出**，或首页 **导出中心**。

第二步：勾选要导出的 **会话**（可多选）。

第三步：选择 **时间范围** 与 **导出格式**（以界面实际选项为准）。

第四步：设置 **输出目录**，点击创建/开始导出，等待进度完成。

第五步：到输出文件夹查看 HTML 等成品；导出 ChatLab 格式后可配合 [ChatLab](https://chatlab.fun/) 做进一步分析。

![导出-选择会话](images/10-export-sessions.png)

## 4、导出页自动化任务

**适用场景**：WeFlow **长期保持打开** 时，按固定 **时间间隔** 自动重复导出（如每隔数小时导出有新消息的会话）。

**使用教程**

第一步：进入 **导出** 页面，打开 **自动化导出** 面板（或「新建任务」入口）。

第二步：在导出页先勾选目标会话，点击 **加入任务**，再设置 **间隔天数/小时**、时间范围模板等。

第三步：保存并 **启用** 任务；应用运行期间调度器会周期性检查并执行。

**与「定时导出」的区别**：

| 方式 | 入口 | 适合 |
|------|------|------|
| **导出页自动化** | 导出 → 自动化 | 应用开着、HTML 等完整导出模板 |
| **设置 → 定时导出** | 设置内多任务 + 系统计划任务 | 每天固定时刻、Markdown/JSON、脚本/Agent |

![导出-自动化](images/11-export-automation.png)

## 12、Agent 连接（本地 API）

**适用场景**：让 Codex、Cursor、n8n、自研脚本通过 HTTP 读取本地聊天记录；也是 **定时导出** 的前置条件。

**使用教程**

第一步：打开 **设置** → **Agent 连接**（原「API 服务」）。

第二步：打开 **本地 API 服务** 开关，确认状态为 **运行中**。

第三步：确认 **监听地址**（默认 `127.0.0.1`，仅本机访问；容器场景可改为 `0.0.0.0` 并务必设置 Token）。

第四步：设置 **服务端口**（默认 `5031`）。

第五步：在 **Access Token** 处填写或点击 **随机生成** 鉴权凭证；外部请求需携带 `Authorization: Bearer <token>` 或 `?access_token=`。

第六步：复制界面上的 **API 地址** 供脚本使用；需要推送新消息时可开启 **主动推送** 并复制 SSE 地址。

第七步：在 Agent 连接页可查看常用 API 路径说明；外部集成请使用 Bearer Token 访问 `http://127.0.0.1:5031` 下的接口。

**安全提示**：启用后，本机其他程序在获得 Token 后可能读取你的聊天记录，请勿将 Token 提交到公开仓库。

## 13、定时导出（多任务）

**适用场景**：为多个群聊分别配置「每天导出昨天」「指定某一天」等规则，并可用 Windows 计划任务在固定时刻自动执行。

**使用教程**

**在应用内配置（推荐）**

第一步：确保已完成 [Agent 连接](#12-agent-连接本地-api)，且 API 为 **运行中**。

第二步：**设置** → **定时导出**。

第三步：点击 **新建任务**，填写 **任务名称**。

第四步：在 **选择群聊** 区域搜索群名，从列表中 **点击选择真实群聊**（显示群名与 `xxx@chatroom` ID）。

第五步：设置 **时间范围**：昨天 / 今天 / 指定日期（YYYYMMDD）/ 最近 N 天。

第六步：选择 **导出格式**：Markdown + 图片 / JSON / ChatLab JSON；按需勾选 **导出媒体**（图片、语音等）。

第七步：可选填写 **输出目录**；留空则使用默认目录。

第八步：点击 **保存任务**；在列表中可 **启用/停用**、**编辑**、**删除**、**预览** 或 **立即运行** 单条任务。

第九步：点击 **打开配置目录** 可查看 `weflow-export-schedule.config.json`（位于 `文档\WeFlow\`）。

**配置 Windows 计划任务（每天自动跑）**

第一步：在 **设置 → 版本更新** 中开启 **开机自启动**，确保凌晨触发时 WeFlow 已运行。

第二步：打开「任务计划程序」→ 创建基本任务，触发器设为每天凌晨（如 01:00），与任务中 `dateMode: 昨天` 配合可导出前一天记录。

第三步：在应用内 **设置 → 定时导出** 配置好任务后，触发时刻在 WeFlow 中点击 **立即运行** 验证；若需无人值守，请保持用户已登录且 WeFlow 在运行（本发行包不提供 Node 脚本，以应用内导出为准）。

第四步：确保触发时刻 **WeFlow 正在运行**、已连接数据库，且 **Agent 连接** 为运行中。

详细说明见 [SCHEDULED-EXPORT.md](SCHEDULED-EXPORT.md)。

![定时导出-选择群聊](images/17-scheduled-export-picker.png)

**适用场景**：在 **Codex**、**Cursor** 等 AI 助手中，用自然语言配置定时导出、生成计划任务命令、排查 API 连接问题。

| 工具 | 路径 |
|------|------|
| Cursor | 项目内已自带，或 `~/.cursor/skills/weflow-scheduled-export/` |
| Codex CLI | `~/.codex/skills/weflow-scheduled-export/` |

也可对 AI 说：「请按 WeFlow 项目的 weflow-scheduled-export skill 帮我配置定时导出。」

![Skill-引用](images/18-skill-reference.png)

**共用前置条件**

第一步：WeFlow 已安装并完成 **数据库连接**。

第二步：**设置 → Agent 连接** 中 API 已 **运行中**，且已设置 **Token**。

第三步：需要导出群聊前，先打开一次 **聊天** 页加载会话列表（便于「定时导出」里选群）。

### 让 AI 帮你配置定时导出任务

**使用教程**

第一步：对 AI 说明需求，例如：「帮我在 WeFlow 里配置每天凌晨导出群『项目群』昨天的聊天记录为 Markdown。」

第二步：AI 应引导你确认 Agent 连接、在 **设置 → 定时导出** 中新建任务，或编辑 `Documents/WeFlow/weflow-export-schedule.config.json`。

**使用教程**

第二步：AI 应生成「任务计划程序」步骤：触发时刻保持 WeFlow 运行、在应用内 **定时导出** 已配置任务，或说明通过 Agent API 的 `curl` 健康检查示例。

第三步：你确认触发时间与 **设置 → 版本更新** 中的开机自启是否已开启。

## 常见问题

**无法连接数据库 / 找不到路径？**  
确认微信 4.0+ 已登录过本机；手动选择正确的 `xwechat_files`（或 macOS 对应目录）；Windows 可尝试管理员运行 WeFlow。

**macOS 提示密钥获取失败？**  
见 [MAC-KEY-FAQ.md](MAC-KEY-FAQ.md)。

**Agent 连接 / API 无法访问？**  
确认设置里服务为 **运行中**；检查端口是否被占用；防火墙是否拦截；Token 是否正确。

**定时导出里群聊列表为空？**  
先打开 **聊天** 页等待会话加载完成，再回到 **设置 → 定时导出** 新建任务。

**计划任务执行了但没有文件？**  
确认触发时刻 WeFlow 在运行、已连库、API 已开启；查看任务输出目录下的 `export.log`。

**SOP / AI 整理失败？**  
检查 **设置 → AI 设置 → 基础配置** 的 API 地址、Key、模型；确认群聊在时间范围内有消息。

**如何联系客服？**  
特供版首页标注微信客服：**sihai332211**（以实际界面为准）。

**如何更新软件？**  
**设置 → 版本更新** 中检查更新，或从 [WeFlow-Release](https://github.com/NewbieCheng/WeFlow-Release/releases/latest) 下载新版本安装。

### 支持平台与安装包

| 平台 | 架构 | 安装包 |
|------|------|--------|
| Windows | x64 | `.exe` |
| macOS | Apple Silicon (arm64) | `.dmg` |
| Linux | x64 | `.AppImage`、`.tar.gz` |

### 常用导出格式

| 场景 | 格式 | 入口 |
|------|------|------|
| 课程资料 Word | docx | 聊天整理 / SOP |
| 归档 Markdown | md | SOP / 定时导出 |
| 网页归档 | HTML | 导出中心 |
| 数据分析 | JSON / ChatLab | 导出中心 / 定时导出 |
| 表格 | Excel / CSV | 导出中心 |

### 重要本地路径（Windows 示例）

| 路径 | 内容 |
|------|------|
| `文档\WeFlow\` | 应用数据根目录 |
| `文档\WeFlow\ai-sop\` | SOP / 聊天整理输出 |
| `文档\WeFlow\weflow-export-schedule.config.json` | 定时导出任务配置 |
| `文档\WeFlow\exports\` | 导出中心产物（视设置而定） |
| `文档\WeFlow\Images\` | 解密图片缓存 |

### 设置页 Tab 对照

| 设置项 | 说明 |
|--------|------|
| 外观 | 主题、明暗模式 |
| 通知 | 桌面通知与过滤 |
| 防撤回 | 撤回消息本地保留 |
| 数据库连接 | 路径、wxid、密钥 |
| 模型管理 | 语音转写模型 |
| 自动下载 | Windows 原图自动下载 |
| 缓存 | 缓存目录 |
| **Agent 连接** | 本地 HTTP API、Token、推送 |
| **定时导出** | 多群、多日定时归档任务 |
| 分析 | 统计相关选项 |
| 安全 | 应用锁 |
| 版本更新 | 检查更新、自启动 |
| AI 设置 | 基础 AI、见解、足迹、**聊天整理** |
